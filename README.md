## 🚀 Deployment

This site is hosted via **GitHub Pages**.

To deploy your own copy:
1. Fork this repository
2. Go to `Settings` → `Pages`
3. Set source to `main` branch / `root`
4. Your site will be live at `https://<your-username>.github.io`
